Description

In this stage, we will save our edited photo - the save button comes into play.
The user should be able to save loaded photo,  so that it will appear in the gallery folder.

Objectives

Add the following components to the application layout:

  1. Button for saving your photo with the id attribute btnSave

 

Then you need to add a proper onClick method to btnSave . This method should save the current photo to the  "/Pictures" folder in
Android default format/JPEG format. 

Basically what main test check is if btnSave  saves some file in "folder":

MediaStore.Images.Media.EXTERNAL_CONTENT_URI

if there is a file it's check it's bitmap.
