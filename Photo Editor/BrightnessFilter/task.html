Description

In this stage, you will implement our first filter - brightness filter.
Image brightness is simply the average RGB value - (R+G+B)/3
To increase/decrease it simply add/subtract the same amount from each rgb color.
We will regulate image brightness with a slider.

Objectives

Add the following components to the application layout:

  1. Slider widget for specifying the photo brightness with the id attribute slBrightness; the values range from -250 to 250 with the step size 10 and initial value 0

Then attach onChangeListener to this slider, that will apply brightness filter.
The Brightness filter should simply add the current value of brightnessSlider to each RGB value of the default photo bitmap and check whether the new value is in the correct boundaries (0,255).
Remember that every time a user loads a new image you must save it's default Bitmap and operate on it while applying a brightness filter (not on current Bitmap, because filter will erase image details).
